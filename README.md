# CIS667_project

## dependencies
- `random`
- `numpy`
- `torch`
- `matplotlib.pyplot`

## How to run interactive domain program
`python3 play_checker.py`

## How to evaluate minimax 
`python3 evaluate_minimax_checker.py`

## How to evaluate neural network
`python3 evaluate_nn_checker.py`
